Fitness Freak is a clean, modern Android fitness tracker MVP built with Flutter (Dart) and Riverpod, featuring Google sign-in via Firebase Authentication, step tracking (pedometer + optional Health Connect), workout logging stored in Firestore, calories burned estimation, and a daily/weekly dashboard. It also includes activity recognition, GPS run tracking, and smart reminders (server heuristics with optional OpenAI). A lightweight Node.js + Express + PostgreSQL backend stores synced daily aggregates and serves stats/reminder suggestions. The UI follows a purple / lavender / white / green theme.
